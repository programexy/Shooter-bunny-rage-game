# Shooter-bunny

This game is a really interesting game.
Although it is a really extremely hard game, you should still try it.
This is a rage game.
Instrctions will be included in the game.
This game is still in beta, so some options are not yet finished.
However, so far ALL of the levels are possible.

The insturctions are:
1. Arrow keys to move and jump.
2. Space to shoot
3. All levels are possible
4. Total of nine levels will be done
5. Have fun!
